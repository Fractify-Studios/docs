## Requirements

- [ox_lib](https://github.com/overextended/ox_lib/releases)
- [ox_mysql](https://github.com/overextended/oxmysql/releases)
- [es_extended](https://github.com/mitlight/es_extended)

## Installation

<Steps>
<Step>
 
### Install requiements

Install all of the requirements listed above.

</Step>
 
<Step>
 
### Download latest fr_garages release

Head over to the [releases](https://github.com/Fractify-Studios/fr_garages/releases) page and download the latest release.

<Callout>
If you are upgrading from a previous version of fr_garages, make sure all old files are deleted.

</Callout>
 
</Step>

<Step>

### Move files to server

Unzip and move the script folder to your `resources` folder.

</Step>

<Step>

### Start script with the server

Add the following line at the end of the `server.cfg` file.

```lua
ensure fr_garages
```

</Step>

</Steps>
